# Pad2Input
A cross-platform reimagination of 00luca00's "Pad2Mouse" designed to act as an alternative for platforms that don't natively support XInput. P2I is **not** a replacement of P2M as there are things that one does that the other doesn't and vice versa.

## Differences between Pad2Mouse and Pad2Input
| Function                                 | Pad2Mouse          | Pad2Mouse          |
|------------------------------------------|--------------------|--------------------|
| Cursor movement with analog stick        | ✅ Working         |                    |
| Left / right mouse click                 | ✅ Working         |                    |
| Xbox controller support                  | ✅ Working         |                    |
| PlayStation controller support           | ⚠️ Not implemented |                    |
| Scrolling support                        | ⚠️ Not implemented |                    |
| On-screen configuration UI               | ⚠️ Not implemented |                    |
